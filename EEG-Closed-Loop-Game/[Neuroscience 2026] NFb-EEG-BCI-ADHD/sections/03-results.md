# 03 - Results

## 3.1 EEG Attention Scoring

### 原文关键数据

| 指标 | 对照组 (Round 1 → 2) | ADHD组 (Round 1 → 2) |
|------|----------------------|---------------------|
| Pre-game | 51.20 → 56.50 | 37.67 → 42.96 |
| In-game | 81.15 → 84.22 | 62.84 → 70.75 |
| Post-game | 56.91 → 59.63 | 44.52 → 51.14 |
| Overall | 63.09 → 66.78 | 48.34 → 54.95 |

- Round 1（标准训练）：注意力 +14.68%（ADHD +21.5%, 对照 +7.85%）
- Round 2（个性化后）：再 +10.05%（ADHD +6.7%, 对照 +13.4%）
- 组间差距从 Round 1 的 30.5% 缩小到 Round 2 的更小值

### 批注

#### ⚠️ 练习效应未控制
- Round 1 和 Round 2 是**顺序进行**的，没有随机化或 counterbalance
- 个性化的"额外效果"可能部分来自**练习效应**（被试第二轮更熟悉游戏）
- 作者辩称"如果是练习效应，对照组应该提升更多" — 有一定道理但不严密
- 更好的做法：加 **sham 反馈对照组**（接收他人的 EEG 反馈而非自己的）

#### ℹ️ 趋势有意义
- ADHD 组获益比例更大，与临床直觉一致（低基线 → 更大改善空间）
- 但注意力值全部基于 NeuroSky eSense 黑盒指标

## 3.2 Behavioral Data

### 原文关键数据

| 指标 | 对照组 (R1 → R2) | ADHD组 (R1 → R2) |
|------|-------------------|-------------------|
| 时间分数 | 4.11 → 4.37 | 3.25 → 3.93 |
| 准确率分数 | 4.32 → 4.51 | 3.64 → 4.05 |
| 游戏分数 | 3.88 → 4.18 | 3.28 → 3.84 |
| 总分 | 12.31 → 13.06 | 10.17 → 11.82 |

- 整体改善 19%（对照 +15.9%, ADHD +22.0%）
- 按键准确率 +3.125%
- 游戏分数 +48.23%（ADHD +56.93%!）
- 完成时间 -5.51%

### 批注

#### 👍 行为数据比 EEG 数据更有说服力
- 行为指标是客观可测量的，不依赖黑盒算法
- 0-5 分制量化虽非标准认知量表，但在游戏场景下合理

#### ⚠️ 缺失分析
- **行为数据与 EEG 注意力指数的相关性没有直接分析**
- 如果两者高度相关 → XGBoost 的高 R² 就更不意外（游戏参数 → 行为 → eSense 可能有因果链）
- 如果两者独立改善 → 需要讨论为什么，以及哪个更可靠

## 3.3 ML Model Comparison

### 原文关键数据

| Model | Variance | MAE | RMSE | R² |
|-------|----------|-----|------|----|
| SVR | 0.7950 | 1.9937 | 2.5852 | 0.8949 |
| **XGBoost** | **0.9743** | **0.6417** | **0.8560** | **0.9826** |
| MLP | 0.7071 | 2.9783 | 3.5127 | 0.9015 |

MLP 深度 ablation：2层 R²=0.895, 3层 R²=0.902, 4层 R²=0.885

### 批注

#### 🔴 MLP 的指标自相矛盾
- MLP R² = 0.9015 但 RMSE = 3.5127
- SVR R² = 0.8949 但 RMSE = 2.5852
- **R² 更高但 RMSE 也更高** — 这说明 MLP 的预测方差很大，可能有极端异常值
- 作者完全没有解释这个矛盾
- 可能原因：MLP 在某些被试上过拟合严重，在其他被试上欠拟合

#### ⚠️ R² 可信度
- 如前所述，窗口级随机 split 下的 R² = 0.9826 几乎肯定被高估
- sensitivity analysis（60:40 和 80:20 split）只是改变比例，没有改变"窗口级随机"的本质问题

#### 👍 Feature Importance
- 滑动加速度 > 转弯速度 > 障碍物数量 > 重力
- 直觉上合理：速度相关参数对注意力要求更高
- 最优注意力区间：转弯速度 300-350，加速度 350-400，障碍物 12-15，重力 2.0-2.5
- 符合 attentional load theory（适度挑战不过载）
