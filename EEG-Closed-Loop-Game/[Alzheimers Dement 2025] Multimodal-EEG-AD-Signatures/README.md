## 一句话总结

系统综述 49 篇多模态研究（EEG/MEG + PET/CSF 蛋白标志物），追踪 AD 全病程电生理特征的**非单调演化**，核心发现：**α振荡在 AD 早期先代偿性增强后再下降**（双相轨迹），不同频段与 Aβ/tau 病理有不同的关联模式。

---

## 核心贡献

1. **首次系统整合 EEG+分子标志物**：筛选 2376 篇→ 最终 49 篇纳入（同时有 EEG/MEG + Aβ/tau 蛋白水平）
2. **揭示 Alpha 双相轨迹**：早期代偿性↑ → 晚期退行性↓（非简单线性下降！）
3. **频段-病理特异性**：δ/θ 与 Aβ 更相关，α/β 与 Aβ+tau 双重相关
4. **Meta-analysis 验证**：Forest plot 按疾病阶段分组，验证了 α 活动的非单调性

### Table 1: 频谱特征汇总

| 频段 | AD/MCI 变化 | 关键发现 | 引用数 |
|------|------------|----------|--------|
| **δ (1-4Hz)** | ↑ 增加 | 直接与 AD 病理相关；在 SNAP 中反而下降 → AD 特异性 | 10篇 |
| **θ (4-8Hz)** | ↑ 增加 | 角色不明确；可能早期代偿性↑，也关联非 AD 病理（SNAP） | 10篇 |
| **α (8-12Hz)** | **双相**: 先↑后↓ | 早期代偿性增强（与 Aβ 沉积相关），后期下降（神经退行） | 8篇↓ + 4篇↑ |
| **β (12-30Hz)** | ↓ 降低 | AD 特异性，枕/颞/顶区显著；SNAP 中 β 反而增加 | 7篇 |
| **γ (30-100Hz)** | ? 不一致 | 有增有减，AD 特异性不确定 | 4篇 |

## 核心创新：Alpha 双相轨迹

```
AD 进展 ──────────────────────────────────────────▶

α 活动:     ___/\___
           /        \
          /          \
         /            \
    基线              \
                       \___________

        临床前     MCI      轻度AD     中重度AD
        (Aβ+)    
        
说明:
- 临床前期：Aβ 开始沉积 → 抑制性中间神经元受损 → E/I 失衡 → α 代偿性增强
- MCI/轻度AD：tau 病理加速 → 神经退行 → α 开始下降
- 中重度AD：广泛神经丢失 → α 严重衰减 + δ/θ 大幅增加

⚠ 这意味着：
- 不能简单用"α↓ = AD"做早期筛查
- 早期 Aβ+ 个体可能 α 反而更高
- 需要纵向追踪，结合分子标志物
```

### 机制解释
1. **E/I 失衡假说**：Aβ 优先损伤 GABA 能抑制性中间神经元 → 去抑制 → 皮层过度兴奋
2. **胆碱能退化**：Aβ 减少 ACh 合成 → 低频活动(δ/θ)增加
3. **突触毒性级联**：AβO → NMDA 过度激活 → Fyn/tau → 突触蛋白丢失 → 突触功能障碍
4. **tau 驱动的退行**：tau 异常磷酸化 → 微管解体 → 线粒体运输障碍 → 氧化应激 → 神经元死亡

## 局限与警示

1. **α 非单调性**：训练 ML 分类器时若不考虑疾病分期，会过拟合
2. **θ 角色不确定**：可能代偿也可能病理，需结合 Aβ 状态判断
3. **γ 不一致**：需更多研究
4. **纵向研究稀缺**：大部分是横断面研究
5. **血浆标志物**：有前景但灵敏度仍不够

## 对脑控游戏项目的关键启示

### 1. 指标选择需考虑疾病分期
- **MCI/轻度 AD**（游戏干预的主要目标人群）：α 可能正常甚至偏高 → 不能简单用 "α↓" 做监测
- **更好的综合指标**：SPR = (α+β)/(δ+θ) 或 POSI → 整合多频段信息
- **APF（峰频率）**比 α 功率更稳定：即使功率代偿性升高，APF 仍然下移

### 2. 不同频段反映不同病理过程
- δ/θ ↔ Aβ 病理为主 → 可作为"病理负荷"指标
- α/β ↔ Aβ + tau 双重 → 反映"神经退行程度"
- 游戏干预如果有效：预期 α/β 恢复 > δ/θ 降低

### 3. AEC α/β 降低出现在认知下降之前
- → 可作为**早期预警**指标
- 但实时计算复杂度较高，适合会话前后评估

### 4. 微状态 C 与认知评分相关
- → 可作为辅助疗效评估指标
