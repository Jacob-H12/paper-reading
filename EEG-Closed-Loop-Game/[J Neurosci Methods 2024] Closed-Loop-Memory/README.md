# Decoding EEG for Optimizing Naturalistic Memory

**作者**: Jeremy H. Rudoler et al.  
**期刊**: Journal of Neuroscience Methods, 410:110220 (2024)  
**DOI**: 10.1016/j.jneumeth.2024.110220  
**PMID**: 39033965

## 一句话总结

构建实时 EEG 解码平台，预测记忆编码状态（好/差），在闭环实验中选择性地在预测好记忆状态时呈现学习材料，操控记忆编码时机以优化自然记忆表现。

## 核心贡献

1. **实时记忆状态解码**: 从 EEG 特征实时预测当前时刻的记忆编码质量
2. **闭环记忆操控**: 根据解码结果选择性呈现材料（好状态呈现 vs 差状态呈现）
3. **自然记忆范式**: 不同于传统实验室范式，使用更生态化的记忆任务
4. **平台化设计**: 可复用的闭环 EEG 实验平台
