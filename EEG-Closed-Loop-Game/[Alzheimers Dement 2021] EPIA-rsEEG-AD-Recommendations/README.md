## 共识声明（3条核心）

### 共识 1：标准化
- 统一患者指导语（闭眼静息、保持清醒）
- 标准化 rsEEG 记录方法（≥19 通道，10-20 系统，≥5 分钟）
- 标准化伪迹检测与剔除

### 共识 2：推荐指标
| 类别 | 频段 | 指标 | 用途 |
|------|------|------|------|
| 功率谱 | **δ (< 4 Hz)** | 功率密度 ↑ | 疾病进展标志 |
| 功率谱 | **θ (4-8 Hz)** | 功率密度 ↑ | 疾病进展标志 |
| 功率谱 | **α (8-12 Hz)** | 功率密度 ↓ / 峰频率 ↓ | 最稳定的 AD 标志 |
| 连接性 | δ/θ/α | DTF / PLI / LLC 等 | 网络退化标志 |

### 共识 3：国际多部门合作
- 需要大规模多中心验证
- 需要监管机构认可
- 呼吁学术-工业-监管三方合作

## 关键发现（文献综述部分）

### Alpha 节律（最一致的发现）
- **后部 Alpha 功率降低**：AD 最经典的 EEG 标志，跨几十年文献一致
- **Alpha 峰频率减慢**：从正常 ~10Hz 降至 8-9Hz
- **机制**：反映丘脑-皮层"断连模式"（thalamocortical disconnection mode）
  - 正常：Alpha 节律由皮层-丘脑-基底前脑-脑干网络调控，反映全局唤醒/意识状态
  - AD：Alpha 振幅降低 = 持续性背景去同步化

### Delta/Theta 慢波增加
- **广泛分布的 δ/θ 功率增加**
- 与疾病严重程度正相关
- 反映"频谱慢化"（spectral slowing）

### 连接性降低
- **Alpha 频段 interrelatedness 降低**
- 反映长程皮层连接退化
- 多种算法一致：DTF, PLI, LLC, coherence

### 与认知评分相关
- Alpha 功率/连接性与 MMSE 分数正相关
- δ/θ 功率与 MMSE 分数负相关

## 生理学模型（Figure 4）

```
正常老年人 (CU):
  皮层-丘脑-基底前脑-脑干 网络 → 主导 Alpha 节律 (~10Hz)
  谷氨酸能 / 胆碱能 / 多巴胺能 / 5-HT 能系统 → 调节唤醒和意识

AD 患者:
  Alpha 振幅 ↓↓↓ (持续性背景去同步化)
  Delta/Theta 振幅 ↑↑↑ (病理性慢化)
  = 丘脑-皮层 "断连模式"
  → 胆碱能系统退化 → ACh 合成减少 → 低频活动增加
```

## 方法学建议

| 项目 | 建议 |
|------|------|
| 通道数 | ≥19（10-20 系统），推荐 64+ |
| 记录时长 | ≥5 分钟闭眼静息 |
| 采样率 | ≥256 Hz |
| 参考电极 | 报告使用何种参考，推荐 eLORETA 源重建 |
| 伪迹处理 | ICA 去除眼动/肌电/心电 |
| 频段定义 | δ < 4Hz, θ 4-8Hz, α 8-12Hz, β 12-30Hz |
| 个体化 | 推荐使用个体 Alpha 频率（IAF）定义频段边界 |

---

## 对脑控游戏项目的启示

1. **指标选择有权威背书**：Alpha 功率 + 峰频率 + δ/θ 功率 是专家共识推荐的一级指标
2. **标准化很重要**：游戏前的 baseline 记录需严格按照共识方法执行
3. **连接性指标**为二级推荐：可作为辅助评估但实时计算难度较大
4. **个体化 Alpha 频段**：用 IAF 代替固定 8-13Hz，对 AD 患者尤其重要（因为 APF 已经下移）
5. **丘脑-皮层断连模型**：为游戏干预提供靶点 — 训练目标是恢复 Alpha 节律和丘脑-皮层同步
