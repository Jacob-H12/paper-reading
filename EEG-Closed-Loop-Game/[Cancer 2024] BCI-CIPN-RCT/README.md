# Brain-Computer Interface Relieves Chronic Chemotherapy-Induced Peripheral Neuropathy: A Randomized, Double-Blind, Placebo-Controlled Trial

**作者**: Sarah Prinsloo, Ted J. Kaptchuk, Dirk De Ridder, Robert Lyle, Eduardo Bruera, Diane Novy, Carlos H. Barcenas, Leonardo G. Cohen  
**期刊**: Cancer, 130(2):300-311 (2024)  
**DOI**: 10.1002/cncr.35027  
**PMID**: 37733286  
**开放获取**: ✅

## 一句话总结

双盲 RCT 验证 EEG 闭环 BCI 游戏训练（20 次）治疗化疗诱导的周围神经病变（CIPN），BCI 组效应量 1.07（vs 候补对照），且 1 个月随访后症状继续改善（安慰剂组停滞），脑电变化可预测 1 个月后症状。

## 核心贡献

1. **高质量 RCT 设计**: 双盲 + 安慰剂对照 + 候补对照，三组比较
2. **EEG 闭环游戏**: 玩家通过调节自身 EEG 信号控制视频游戏（音视觉奖励）
3. **持续效果**: BCI 组 1 个月后继续改善，安慰剂组效果停滞
4. **神经预测指标**: 治疗结束时的脑电变化可预测 1 个月后症状（仅 BCI 组）
