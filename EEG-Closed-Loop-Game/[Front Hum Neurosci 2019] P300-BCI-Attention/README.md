# A P300-Based Brain-Computer Interface for Improving Attention

**作者**: Mahnaz Arvaneh, Ian H. Robertson, Tomas E. Ward  
**期刊**: Frontiers in Human Neuroscience, 12:524 (2019)  
**DOI**: 10.3389/fnhum.2018.00524  
**PMID**: 30662400  
**开放获取**: ✅

## 一句话总结

首次将 P300 Speller BCI 改造为自适应难度的神经反馈训练游戏，根据用户 P300 幅度动态提升难度（要求产生更大更强的 P300），28 名被试 30 分钟训练后 P300 增强 + Alpha 抑制增加 + 空间注意力反应时间显著缩短。

## 核心贡献

1. **P300 自适应闭环**: 根据 P300 总能量动态调难度，无需冗长试错校准
2. **双重 ERP 效应**: 目标试次 ERP 增强 + 非目标试次 ERP 衰减
3. **Alpha 抑制迁移**: 训练期间和训练后空间注意力任务中均观察到中央-顶叶 Alpha 抑制
4. **快速起效**: 仅 ~30 分钟训练即产生显著注意力改善
