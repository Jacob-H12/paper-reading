# EEG-Based Closed-Loop Neurofeedback for Attention Monitoring and Training in Young Adults

**作者**: Bingbing Wang, Zhihua Xu, Ting Luo, Jing Pan  
**期刊**: Journal of Healthcare Engineering, 2021:5535810 (2021)  
**DOI**: 10.1155/2021/5535810  
**PMID**: 34234929  
**开放获取**: ✅

## 一句话总结

提出改进随机森林 (IRF) 算法实时分类 5 级注意力水平（平均准确率 79.34%），开发 3 种严肃游戏（拔河/小鸟冒险/贪吃果冻）分别训练持续注意/选择性注意/聚焦注意，10 名被试训练后 Schulte 测试显著提升。

## 核心贡献

1. **5 级注意力实时分类**: IRF（网格搜索 + 多重交叉验证）优于 SVM/KNN/AdaBoost/ET/RF
2. **3 种针对性严肃游戏**: 拔河（持续注意）、小鸟冒险（选择性注意）、贪吃果冻（聚焦注意）
3. **闭环神经反馈**: 游戏难度根据注意力等级实时调整
4. **4 指标验证**: Schulte 时间 + 胜场 + 游戏分数 + 技能时间
