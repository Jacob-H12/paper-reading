# AD 患者稳定 EEG 指标 — 文献调研总结

> 基于两篇核心文献 + 领域知识的系统梳理
> 
> 文献 1: **Babiloni et al. (2021)** — 60+专家共识，rsEEG 用于 AD 临床试验的推荐指标
> - Alzheimers Dement, 17(9):1528-1553. PMID: 33860614
> 
> 文献 2: **Sohrabpour et al. (2025)** — 49篇多模态系统综述，追踪 AD 电生理特征演化
> - Alzheimers Dement, 21(11):e70835. PMID: 41216912

---

## ⚠ 重要前置发现：Alpha 振荡的非单调演化

Sohrabpour (2025) 的核心突破发现：

**α 振荡在 AD 全病程中呈双相（倒 U 型）轨迹：**
- **临床前期**（Aβ+ 但无认知症状）：α 活动**代偿性增强**
- **MCI → 轻度 AD**：α 开始下降
- **中重度 AD**：α 严重衰减

**机制**：Aβ 优先损伤 GABA 能抑制性中间神经元 → E/I 失衡 → 早期过度兴奋/代偿 → tau 加速后神经退行 → 活动下降

**启示**：不能简单用 "α 功率 ↓ = AD"，需结合疾病分期和综合指标。

## 一、确定指标（3-5个）：高稳定性、强效应量、专家共识支持

### ✅ 指标 1: 频谱慢化 — Alpha 功率下降 + Delta/Theta 功率增加

| 维度 | 详情 |
|------|------|
| **状态** | 静息态（闭眼最佳）|
| **现象** | 后部 α(8-12Hz) 功率↓，δ(<4Hz)/θ(4-8Hz) 功率↑ |
| **共识级别** | ⭐⭐⭐⭐⭐ — Babiloni (2021) 专家共识一级推荐 |
| **文献支持** | Sohrabpour (2025): 10+篇独立报告 δ/θ↑，8+篇报告 α↓ |
| **效应量** | α↓: d=0.8-1.5; δ↑: d=0.8-1.2 (中晚期) |
| **稳定性** | 极高 — 40+年文献一致，跨设备跨人群复现 |
| **分子关联** | δ/θ 与 Aβ 沉积正相关；α 与 Aβ+tau 双重相关 |
| **认知关联** | δ/θ↑ ↔ MMSE↓; α↓ ↔ MMSE↓ |
| **注意事项** | **MCI/早期 AD 的 α 可能代偿性升高**（Sohrabpour 2025）|
| **实时提取** | FFT/Welch, 1-2秒窗口, 延迟极低 |
| **游戏映射** | α 功率 → "能量/警觉条"；δ/θ → "疲劳/病理负荷" |

**Babiloni (2021) 的机制模型**：
- 正常: 皮层-丘脑-基底前脑-脑干网络 → 主导 α 节律 → 调节唤醒/意识
- AD: α 振幅↓ = 丘脑-皮层"断连模式" + δ/θ↑ = 病理性慢化
- 胆碱能退化 → ACh 合成↓ → 低频活动↑

---

### ✅ 指标 2: SPR / POSI — 频段功率比综合指标

| 维度 | 详情 |
|------|------|
| **状态** | 静息态 + 任务态均可 |
| **定义** | SPR = (α+β功率) / (δ+θ功率)；POSI = 病理性振荡慢化指数 |
| **共识级别** | ⭐⭐⭐⭐ — Sohrabpour (2025) 重点推荐 |
| **文献支持** | SPR 在 Aβ+ aMCI 中显著↓；POSI 与认知负相关 |
| **效应量** | d=1.0-1.8（比单频段更鲁棒）|
| **稳定性** | 极高 — 综合多频段，抵消单频段波动 |
| **优势** | **避免了 α 双相轨迹的陷阱**——即使 α 代偿性↑，如果 δ/θ 也在↑，SPR 仍会下降 |
| **分子关联** | α/δ 比和 α/θ 比均与 Aβ/tau 负相关 |
| **实时提取** | 极简——两组频段功率的比值 |
| **游戏映射** | 核心闭环变量 "综合认知效率指数" |

**为什么比单一 TAR 更好**：
- TAR 只看 θ/α 两个频段
- SPR 整合四个频段，对 AD 更全面
- POSI 还考虑了相对于年龄匹配健康对照的偏移

---

### ✅ 指标 3: Alpha 峰频率 (APF) 减慢

| 维度 | 详情 |
|------|------|
| **状态** | 静息态（闭眼最佳）|
| **现象** | 个体 α 峰频率从正常 ~10-10.5Hz 降至 ~8-9.5Hz |
| **共识级别** | ⭐⭐⭐⭐⭐ — 两篇文献均重点提及 |
| **文献支持** | Sohrabpour: MCI 峰频 9.5Hz vs HC 10.5Hz；Babiloni: 推荐使用 IAF |
| **效应量** | d=0.9-1.3 |
| **稳定性** | 极高 — test-retest ICC=0.85-0.92，是 EEG 最稳定的个体特征 |
| **关键优势** | **即使 α 功率代偿性升高，APF 仍然下移** → 比 α 功率更稳健的早期指标 |
| **分子关联** | 与 PET Aβ/tau 均负相关；与 CSF Aβ42 正相关、CSF p-tau 正相关 |
| **认知关联** | 与认知处理速度强相关 |
| **实时提取** | Welch PSD → 8-13Hz 峰值检测，非常稳定 |
| **游戏映射** | 绑定游戏节奏/速度；长期改善的标志 |

---

### ✅ 指标 4: P300 潜伏期延长 + 振幅下降

| 维度 | 详情 |
|------|------|
| **状态** | 任务态（oddball 范式）|
| **现象** | P300 潜伏期延长 30-80ms，振幅降低 2-5μV |
| **共识级别** | ⭐⭐⭐⭐⭐ — 经典认知 ERP，几十年文献一致 |
| **效应量** | 潜伏期 d=1.0-1.5, 振幅 d=0.8-1.2 |
| **稳定性** | 极高 — Polich (2007) 综述确认跨数十年复现 |
| **优势** | 任务态指标，直接反映"注意力分配/目标检测"能力 |
| **认知关联** | 与注意力、工作记忆、反应速度强相关 |
| **实时提取** | 需 epoch 平均或 ML 单试次检测（xDAWN），延迟 1-3s |
| **游戏映射** | 天然适配目标检测游戏（"找不同"）；P300 改善 = 通关条件 |

**与游戏行为的天然联系**：
- P300 本质 = "检测到新异/目标刺激"的脑反应
- 游戏中嵌入 oddball 元素（突然出现的特殊物品）→ 直接诱发和监测 P300
- 训练后 P300 改善 = 注意资源分配能力提升

### ✅ 指标 5: Beta 功率降低

| 维度 | 详情 |
|------|------|
| **状态** | 静息态 |
| **现象** | 枕/颞/顶区 β(12-30Hz) 功率显著降低 |
| **共识级别** | ⭐⭐⭐⭐ — Sohrabpour (2025): 7篇报告, AD 特异性 |
| **效应量** | d=0.7-1.0 |
| **稳定性** | 高 — 方向一致，SNAP 中 β 反而增加 → AD 特异性好 |
| **分子关联** | 与 Aβ 正相关/反向 U 型；与 tau 负相关；与 CSF 指标一致 |
| **认知关联** | β↓ ↔ 认知评分↓ (正相关) |
| **AD 特异性** | SNAP 患者 β 增加，AD 患者 β 降低 → 鉴别诊断价值 |
| **实时提取** | FFT，简单 |
| **游戏映射** | β 活动 ↔ 主动运动/认知参与 → "参与度"指标 |

## 二、大概率指标（2-3个）：文献较充分但效果或稳定性略弱

### 🔶 指标 6: Alpha 频段功能连接降低 (AEC / Coherence / PLV)

| 维度 | 详情 |
|------|------|
| **状态** | 静息态 |
| **现象** | α/β 频段的 AEC、coherence、PLV 在 AD 中降低 |
| **文献支持** | Sohrabpour (2025) Table 2: 多种算法一致；Babiloni (2021): 推荐 interrelatedness |
| **效应量** | d=0.6-1.0 |
| **稳定性** | 中高 — 方向一致但数值受参考电极/方法影响 |
| **⭐关键发现** | **α/β AEC↓ 出现在认知下降之前** → 早期预警能力 |
| **注意** | 早期 AD 可能出现 α 超同步（代偿），后期才降低 |
| **实时提取** | PLV/AEC 需 2-4s 窗口，计算量中等 |
| **游戏映射** | 多感官整合任务的内在指标 |

---

### 🔶 指标 7: Delta 功率增加（AD 特异性标志）

| 维度 | 详情 |
|------|------|
| **状态** | 静息态 |
| **现象** | δ(<4Hz) 功率在 AD 中增加 |
| **文献支持** | Sohrabpour (2025): 直接与 AD 病理相关；SNAP 中 δ 反而↓ → AD 特异 |
| **效应量** | d=0.5-0.8 (MCI), d=1.0+ (中期 AD) |
| **稳定性** | 中高 — 中晚期稳定，早期不够敏感 |
| **分子关联** | 与 CSF Aβ42 负相关，与 PET Aβ 正相关 |
| **AD 特异性** | 比 θ 更特异：θ 可能出现在 SNAP 中，但 δ↑ 更偏 AD |
| **实时提取** | FFT，极简 |
| **游戏映射** | δ 过多 → 皮层抑制/嗜睡 → 安全守护（暂停/降速）|

---

### 🔶 指标 8: EEG 微状态 (Microstate) 变化

| 维度 | 详情 |
|------|------|
| **状态** | 静息态 |
| **现象** | AD: 微状态 A/B 出现率↑, C/D 出现率↓ |
| **文献支持** | Sohrabpour (2025) Table 2: 与 fMRI 高度一致 |
| **认知关联** | 微状态 C duration 与 MMSE/MoCA 正相关 |
| **稳定性** | 中 — 较新指标，需更多验证 |
| **实时提取** | 计算量中等，但理论框架成熟 |
| **游戏映射** | 可作为阶段性评估指标（会话前后对比）|

## 四、对闭环游戏设计的指导

### 推荐的主控指标组合

**实时闭环用（每 2 秒更新）：**
1. **SPR = (α+β)/(δ+θ)** — 综合认知效率，主控难度调节
2. **APF** — 个体化校准基准 + 长期改善追踪
3. **δ 功率** — 安全守护（疲劳/嗜睡检测）

**任务嵌入用（游戏内）：**
4. **P300** — 目标检测任务表现，作为游戏通关/升级条件

**阶段性评估用（每次会话前后）：**
5. **β 功率** — 参与度/认知活跃度
6. **α/β AEC** — 网络连接完整性（可能是最早的改善信号）

### 关键设计原则

1. **用 SPR 代替单一 TAR/α 功率**：避免 α 双相轨迹的干扰
2. **每次校准**：AD 患者状态波动大，用 IAF 个体化频段边界
3. **分层考虑**：MCI vs 轻度 AD vs 中度 AD 的基线不同，需不同参数空间
4. **纵向追踪**：APF 回升 + SPR 恢复 = 最可信的改善证据
